## How It Works

```
cron (daily) -> fetch_news.py -> RSS feeds (Google News, The Verge, Ars Technica, ...)
                              -> filter by keywords (brands + WiFi standards)
                              -> deduplicate
                              -> generate briefing (briefings/YYYY-MM-DD.md)
                              -> update README.md
                              -> send email notification
```

**Manual run**: `python3 fetch_news.py`
